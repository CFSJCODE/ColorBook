# 🎨 ColorBook v2.0 — Interface Engineering Palette Engine

**ColorBook** é uma ferramenta de curadoria e exploração cromática desenvolvida para engenheiros de interfaces e designers que buscam alta performance e estética refinada. O projeto não é apenas uma galeria, mas um motor de interação que utiliza princípios de **Fluent UI** e **Liquid Glass** para proporcionar uma experiência de usuário imersiva e responsiva.

---

## 🎯 Por que o ColorBook?

Muitas ferramentas de cores focam apenas no visual. O ColorBook foi projetado sob a ótica da **Engenharia de Software**, garantindo que a busca por uma paleta seja tão rápida e eficiente quanto a implementação no código. É a ferramenta ideal para quem precisa de sugestões de cores aplicadas a mockups reais de sistemas.

## ✨ Principais Funcionalidades

- **Curadoria Técnica:** Paletas selecionadas com foco em acessibilidade, contraste térmico e estética industrial/cyberpunk.
- **Copy-to-Code:** Copie valores HEX individuais ou o código CSS completo de gradientes lineares com um clique.
- **Mockups em Tempo Real:** Visualize como a paleta se comporta em componentes de interface (barras de progresso, ícones, cards) através do sistema *Liquid Glass*.
- **Navegação de Alta Produtividade:** Suporte total a atalhos de teclado (estilo Vim) para desenvolvedores que preferem não tirar as mãos do teclado.

## 🛠️ Arquitetura e Performance

- **Vanilla JavaScript Engine:** Implementação sem frameworks pesados para garantir latência zero.
- **DOM Mapping & O(1) Access:** Cache de elementos para evitar buscas repetitivas na árvore DOM.
- **Interação 3D (Tilt Engine):** Algoritmos de inclinação baseados no baricentro dos componentes para feedback tátil visual.
- **Tailwind CSS + ITCSS:** Estilização escalável e manutenção otimizada.

## ⌨️ Atalhos de Teclado

- `/` : Inicia a busca.
- `Esc` : Limpa e sai da busca.
- `j` / `k` ou `setas` : Navega entre as paletas.
